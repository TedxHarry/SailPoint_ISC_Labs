# Test Data

Production-grade synthetic datasets for three deployment scenarios: small (100 users), mid-market (300 users), and enterprise (500 users).

## Available Datasets

### Contoso Ltd (Small Deployment - 100 Users)
Small organization dataset ideal for learning fundamentals and initial configurations.

**Files:**
- `contoso-identities-100.csv` - 100 test identities
- `contoso-finance-app.csv` - Finance application accounts
- `contoso-hris.csv` - HR system accounts

### RegionTech Inc (Mid-Market - 300 Users)
Mid-market organization with more complex structure and multiple integrated systems.

**Files:**
- `regiontech-identities-300.csv` - 300 test identities
- `regiontech-finance.csv` - Finance application accounts
- `regiontech-hris.csv` - HR system accounts
- `regiontech-salesforce.csv` - Salesforce integration accounts

### GlobalBank Corp (Enterprise - 500 Users)
Enterprise-scale dataset with multiple departments and complex governance requirements.

**Files:**
- `globalbank-identities-500.csv` - 500 test identities
- `globalbank-finance.csv` - Finance application accounts
- `globalbank-hris.csv` - HR system accounts
- `globalbank-salesforce.csv` - Salesforce integration accounts
- `globalbank-crm.csv` - CRM system accounts

## How to Use

1. Choose your scenario (Contoso, RegionTech, or GlobalBank)
2. Download the CSV files
3. Import into your ISC environment following the module instructions
4. Use the data for hands-on labs and practice exercises

## Data Privacy

All data is **synthetic** and created for educational purposes only. No real identities or organizations are represented.
