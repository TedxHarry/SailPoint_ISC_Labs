## Domain 1: Fundamentals & Architecture

### Question 1.1
**What is the primary purpose of SailPoint Identity Security Cloud (ISC)?**

A) To replace your Active Directory
B) To provide centralized identity governance and access management
C) To store all user credentials
D) To automatically delete inactive accounts

**Difficulty:** Beginner
**Related Module:** 1.3-introduction-to-sailpoint-isc

---

### Question 1.2
**In the ISC architecture, what role does the Virtual Appliance perform?**

A) It hosts the web UI where users log in
B) It acts as a connector bridge between on-premises systems and ISC
C) It replaces your email server
D) It only connects to Azure

**Difficulty:** Beginner
**Related Module:** 1.3-introduction-to-sailpoint-isc

---

## Answer Key

### Domain 1: Fundamentals & Architecture

**1.1 Answer: B**
- ISC provides centralized identity governance and access management across all systems
- Incorrect answers: A (ISC doesn't replace AD), C (ISC doesn't store credentials), D (ISC can be configured to archive accounts, not auto-delete)

**1.2 Answer: B**
- The Virtual Appliance acts as a secure bridge allowing ISC to connect to on-premises systems
- Incorrect answers: A (web UI hosted on ISC), C (doesn't replace email), D (connects to multiple systems)
