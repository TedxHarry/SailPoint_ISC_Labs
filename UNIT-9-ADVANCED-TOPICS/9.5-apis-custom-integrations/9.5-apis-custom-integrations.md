## Key Concepts

### What Are ISC APIs?

**APIs (Application Programming Interfaces)** allow programmatic interaction with ISC beyond the web UI.

**Use Cases:**
- Custom Integration: Connect ISC to proprietary systems
- Automation: Automate governance workflows beyond ISC native capabilities
- Analytics: Extract data for external BI tools
- Bulk Operations: Manage thousands of records efficiently
- Mobile Apps: Build mobile interfaces to ISC

### ISC API Architecture

**REST API Design:**
```
Base URL: https://your-tenant.identitynow.com/api/v3

Endpoints:
  - /identities
  - /accounts
  - /access-profiles
  - /entitlements
  - /workflows
  - /certifications
  - /audit-log
```

**Example Request:**
```
GET /api/v3/identities?filters=department%20eq%20%22Finance%22
Authorization: Bearer [access_token]
```

**Example Response:**
```json
{
  "count": 45,
  "results": [
    {
      "id": "2c91808475cdf39a0175d15f",
      "name": "Alex Lee",
      "email": "alex.lee@contoso.com",
      "department": "Finance",
      "accountCount": 4
    }
  ]
}
```

### API Authentication Methods

| Method | Use Case | Security |
|--------|----------|----------|
| **OAuth 2.0** | User-facing apps | High (token expiration) |
| **Personal Access Token (PAT)** | Service-to-service | High (long-lived, revocable) |
| **API Key** | Legacy systems | Medium (no expiration) |

#### Step 3: Write Python Integration

**Action:** Create file: `contoso_sync.py`

**Code:**
```python
#!/usr/bin/env python3
"""
Contoso Contractor Sync Integration
Syncs contractor data from HRIS to ISC
"""

import requests
import json
import logging

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

# Configuration
ISC_URL = "https://tenant.identitynow.com/api/v3"
ISC_TOKEN = "eyJhbGciOi..."  # Use environment variable!
HRIS_URL = "https://hris.contoso.com/api/v1"

def get_isc_contractors():
    """Get existing contractors from ISC"""
    headers = {"Authorization": f"Bearer {ISC_TOKEN}"}
    response = requests.get(
        f"{ISC_URL}/identities",
        params={"filters": 'department eq "Contractors"'},
        headers=headers
    )
    response.raise_for_status()
    return {c["email"]: c for c in response.json().get("results", [])}

def sync_contractor(contractor, existing):
    """Sync single contractor to ISC"""
    headers = {"Authorization": f"Bearer {ISC_TOKEN}"}

    if contractor["email"] in existing:
        # Update existing
        identity_id = existing[contractor["email"]]["id"]
        update = {
            "attributes": {
                "contractor_id": contractor["id"],
                "project": contractor["project"],
                "end_date": contractor["end_date"]
            }
        }
        requests.patch(
            f"{ISC_URL}/identities/{identity_id}",
            json=update,
            headers=headers
        )
        logger.info(f"Updated contractor: {contractor['name']}")
    else:
        # Create new
        new_identity = {
            "name": contractor["name"],
            "email": contractor["email"],
            "department": "Contractors",
            "attributes": {
                "contractor_id": contractor["id"],
                "project": contractor["project"],
                "end_date": contractor["end_date"]
            }
        }
        requests.post(
            f"{ISC_URL}/identities",
            json=new_identity,
            headers=headers
        )
        logger.info(f"Created contractor: {contractor['name']}")

def main():
    """Main sync process"""
    logger.info("Starting contractor sync...")

    # Get existing contractors
    existing = get_isc_contractors()
    logger.info(f"Found {len(existing)} existing contractors")

    # Get contractors from HRIS
    response = requests.get(f"{HRIS_URL}/contractors?status=ACTIVE")
    contractors = response.json()
    logger.info(f"Found {len(contractors)} active contractors in HRIS")

    # Sync each contractor
    for contractor in contractors:
        try:
            sync_contractor(contractor, existing)
        except Exception as e:
            logger.error(f"Failed to sync {contractor['name']}: {e}")

    logger.info("Contractor sync complete")

if __name__ == "__main__":
    main()
```

#### Step 5: Error Handling

**Action:** Add error handling to script

**Common Errors:**
```
401 Unauthorized: Token expired or invalid
  - Check token is valid
  - Verify scopes include identities:read

429 Too Many Requests: Rate limit exceeded
  - Implement exponential backoff
  - Wait before retrying

404 Not Found: Resource doesn't exist
  - Check identity ID is valid
  - Verify department filter spelling
```

**Improved Error Handling:**
```python
import time

def sync_with_retry(contractor, existing, max_retries=3):
    """Sync contractor with retry logic"""
    for attempt in range(max_retries):
        try:
            sync_contractor(contractor, existing)
            return True
        except requests.exceptions.HTTPError as e:
            if e.response.status_code == 429:  # Rate limit
                wait_time = 2 ** attempt  # Exponential backoff
                logger.warning(f"Rate limited. Waiting {wait_time}s...")
                time.sleep(wait_time)
            else:
                logger.error(f"HTTP Error: {e}")
                return False
    return False
```

**Expected Result:** Robust error handling added
