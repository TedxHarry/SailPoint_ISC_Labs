## 🔧 TROUBLESHOOTING

**Issue: "Cannot assign manager - user shows but can't select"**
- **Cause:** Users from 2.3 may not be fully synced yet
- **Fix:** Wait a few minutes, refresh page, then try again
- **Prevent:** Create all 10 users first, then assign managers

**Issue: "Error creating user: Username already exists"**
- **Cause:** User name conflicts with 2.3 users or duplicate in this module
- **Fix:** Check Module 2.3 reference for existing names
- **Example:** "Jordan Martinez" (changed from Alex Torres) should not duplicate

**Issue: "Manager assignment won't save"**
- **Cause:** User properties not fully synced
- **Fix:** Move to next user, come back and try again
- **Alternative:** Try using "Edit manager" from user's main profile page

**Issue: "Some users created but not all 10"**
- **Cause:** Process interrupted or page error
- **Fix:** Check user list for which ones are missing
- **Resume:** Create only the missing users (don't re-create existing ones)
