## 📋 Prerequisites

Module 2.1: Entra ID Free Tier Overview. Microsoft account with Entra ID Free.

---

## 📚 HANDS-ON LAB

### Access Azure Portal

1. Go to https://portal.azure.com
2. Sign in with Microsoft account (same account linked to Entra ID)
3. You should see Azure Portal home page

**If you don't have Entra ID yet:**
- Create free Microsoft 365 account, or
- Create Azure free account (includes Entra ID Free)

### Navigate to Entra ID

In Azure Portal:
1. Search bar (top): Type "Entra ID"
2. Click "Microsoft Entra ID" result
3. You're now in Entra ID admin center

### Key Sections You'll Use

**Home:** Overview dashboard

**Manage > Users:** Where you create test users (2.3-2.5)

**Manage > Groups:** Where you create test groups (2.5)

**Manage > App registrations:** Where you register ISC_Test_App (2.6)

**Applications > API permissions:** Where you grant Microsoft Graph permissions (2.7)

**Certificates & secrets:** Where you generate client secret (2.8)
