## 📚 HANDS-ON LAB

### ISC Users (Admin Users)

Note: These are DIFFERENT from Entra ID users. ISC admin users manage the ISC system itself.

In ISC > Administration > Users:

**Create ISC Admin User:**
- Email: [your-email@company.com]
- First name: [your first name]
- Last name: [your last name]
- Role: System Administrator

System Administrator role gives full ISC access.

### Other ISC Roles Available

- System Administrator: Full access
- Administrator: Wide access
- Governance Admin: Governance features only
- Manager: Limited (manage own team)
- User: Self-service access requests only

---

## 🧪 TASK

1. Go to Administration > Users
2. Create yourself as System Administrator (or note if already created)
3. Verify your user exists
4. Test login (you might already be logged in)

## ✅ SUCCESS CRITERIA
