## 🔧 TROUBLESHOOTING

**Issue: "Can't find user to add to group"**
- **Cause:** User not created yet (missing from 2.3 or 2.4)
- **Fix:** Go back and create the missing user
- **Verify:** Check user list in Entra ID > Users first

**Issue: "Group created but members won't add"**
- **Cause:** Members field might be timing out
- **Fix:** Close the group, re-open it, try adding members again
- **Alternative:** Add members one-at-a-time instead of all at once

**Issue: "Member count doesn't match expected"**
- **Cause:** May have missed adding someone
- **Fix:** Open group, check current members against reference table
- **Reference:** Module 2.3 has expected member counts for each group

**Issue: "Can't see newly created group in list"**
- **Cause:** Page not refreshed
- **Fix:** Refresh page (F5) to see new groups
- **Wait:** Sometimes takes 10-30 seconds for group to appear

**Issue: "Group name is wrong (typo)"**
- **Cause:** Mistyped during creation
- **Fix:** Edit group properties to correct name
- **Path:** Entra ID > Groups > [Group Name] > Properties > Name
