# 2.11 - Access ISC Sandbox

**Unit:** Entra ID Setup & ISC Sandbox | **Tier:** 1 | 

---

## 🎯 Learning Objectives

- Log into ISC sandbox
- Understand ISC interface basics
- Verify access

---

## 📋 Prerequisites

Module 2.10: Verify Entra ID Setup. ISC sandbox access credentials (from Identity University enrollment).

---

## 📚 HANDS-ON LAB

### Access ISC Sandbox

**URL:** https://[your-tenant-id].api.identitynow.com

Replace [your-tenant-id] with your ISC tenant identifier (provided in enrollment email).

**Login:**
1. Enter username/email (from enrollment)
2. Enter password (from enrollment)
3. Complete MFA if prompted

**You should see:**
- ISC dashboard with welcome page
- Menu on left side (Administration, Governance, Reporting, etc.)
- Top navigation bar (Identities, Search, etc.)

### If Access Fails

- Confirm enrollment email was received
- Verify correct tenant URL
- Check MFA is set up correctly
- Contact SailPoint if credentials invalid
