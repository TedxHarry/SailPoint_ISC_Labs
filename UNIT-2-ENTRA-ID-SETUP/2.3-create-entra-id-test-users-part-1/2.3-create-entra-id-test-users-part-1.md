## 🔧 TROUBLESHOOTING

**Issue: "User principal name is not available"**
- **Cause:** Username already exists in tenant
- **Fix:** Add a number suffix (e.g., alex.lee2@...)
- **Prevent:** Use full name format (firstname.lastname)

**Issue: "Password must meet complexity requirements"**
- **Cause:** Generated password doesn't meet policy
- **Fix:** Auto-generate password OR use: P@ssw0rd2025!
- **Prevent:** Let system auto-generate passwords

**Issue: "Cannot set manager - user not found"**
- **Cause:** Manager user hasn't been created yet
- **Fix:** Create all users first, then assign managers
- **Note:** If you're in 2.3, skip manager assignment - do it in 2.4

**Issue: "Department attribute doesn't save"**
- **Cause:** May need to refresh page or wait for sync
- **Fix:** Refresh Entra ID page and re-check user properties
- **Prevent:** Wait 10 seconds after creating user before editing properties

**Issue: "Can't see users in list after creation"**
- **Cause:** Page not refreshed, or user creation failed
- **Fix:** Refresh the page (F5), then check if users appear
- **Verify:** Click on user to open details and confirm attributes
