## 📚 HANDS-ON LAB

### Grant Permissions

In ISC_Test_App > API permissions > Add a permission:

**Select Microsoft Graph:**

**Add these permissions (Application type, NOT Delegated):**

1. User.Read.All (Application)
2. Group.Read.All (Application)
3. Directory.Read.All (Application)

**Important:** Select "Application permissions" when adding, NOT "Delegated permissions". ISC Connector requires application-level access to authenticate as a service principal.

**Grant admin consent:**

After adding, click "Grant admin consent for [tenant]"

### Why These Permissions?

- **User.Read.All (Application):** ISC Connector can read all Entra ID users
- **Group.Read.All (Application):** ISC Connector can read all groups
- **Directory.Read.All (Application):** ISC Connector can read directory data

**Why Application, not Delegated?**
- **Delegated:** Requires a signed-in user - won't work for automated connectors
- **Application:** Service-to-service authentication - required for ISC Connector

Without these application-level permissions, ISC connector to Entra ID won't authenticate.
